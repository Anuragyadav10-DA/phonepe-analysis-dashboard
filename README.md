# 📊 PhonePe Transaction Analysis Dashboard

> An interactive Power BI dashboard that transforms PhonePe transaction data into meaningful business insights through data visualization, KPI tracking, and trend analysis.

---

## 📌 Project Overview

The **PhonePe Transaction Analysis Dashboard** is a Power BI project designed to analyze transaction performance, user behavior, and payment trends. The dashboard provides an intuitive and interactive interface that enables users to monitor key business metrics and make data-driven decisions.

This project demonstrates skills in **Power BI, Data Modeling, DAX, Power Query, and Business Intelligence**.

---

## 🎯 Objectives

- Analyze overall transaction performance.
- Monitor payment success rate.
- Track monthly transaction trends.
- Compare transaction values across different services.
- Understand customer age distribution.
- Analyze weekday vs weekend transaction patterns.
- Identify top users by transaction value.
